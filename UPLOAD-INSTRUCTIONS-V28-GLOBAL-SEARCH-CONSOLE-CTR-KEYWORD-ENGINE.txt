GR8 GAMZ V28 upload instructions

Use the update zip, not the full repository, unless something goes wrong.

1. Download the V28 update zip.
2. Unzip it on your computer.
3. Open your GitHub repository root:
   github.com/digitaldownloaden463-afk/gr8gamz-webhub
4. Make sure you are on the main branch.
5. Click Add file > Upload files.
6. Drag the CONTENTS of the extracted V28 folder into GitHub.
7. Do not upload the outer folder itself.
8. Commit directly to main.

Suggested commit message:
Add V28 global Search Console CTR and keyword engine
