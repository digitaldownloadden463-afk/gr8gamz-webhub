# GR8 GAMZ V8 Selection Labs Specification

The Turbo Drift Garage proved that vehicle choice improves replay value. V8 extends that system across the launched games.

## Game systems

### Neon Snake Rush — Snake Lab
Players choose between Neon Viper, Micro Serpent, Turbo Cobra, Crystal Python and Shadow Fang. Each option changes speed, control, combo window or score multiplier.

### Stack Tower Rush — Build Mode
Players choose between Neon Blocks, Wide Starter, Glass Tower, Heavy Slabs and Chaos Blocks. Each mode changes block width, movement speed, forgiveness and score bonus.

### Pixel Goal Hero — Team Locker
Players choose between Classic Striker, Power Boot, Curve Master, Quick Tapper and Captain Shot. Each style changes shot power, accuracy, curve or score multiplier.

### Space Tap Survival — Space Hangar
Players choose between Nova Scout, Tiny Comet, Shield Runner, Rocket Beast and Void Cruiser. Each ship changes hitbox, lift, gravity, starting shield or score multiplier.
