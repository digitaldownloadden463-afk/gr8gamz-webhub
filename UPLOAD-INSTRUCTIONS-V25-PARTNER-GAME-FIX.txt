GR8 GAMZ V25 upload instructions

Use the V25 update zip, not the full repository zip.

1. Download and unzip:
   gr8gamz-v25-partner-game-playability-real-artwork-fix-update.zip

2. Open your GitHub repo:
   github.com/digitaldownloaden463-afk/gr8gamz-webhub

3. Make sure you are on the main branch.

4. Click:
   Add file > Upload files

5. Drag the CONTENTS of the unzipped V25 update folder into GitHub.
   Do not upload the outer folder itself.

Correct paths should look like:
   src/lib/partnerFeedResolver.js
   src/app/api/partner-games/[slug]/route.js
   src/app/more-free-games/[slug]/play/page.js
   src/components/partner/PartnerLiveGamePanel.js
   src/components/partner/PartnerProfileLiveImage.js
   scripts/audit-partner-profiles.mjs

Wrong paths would look like:
   gr8gamz-v25-partner-game-playability-real-artwork-fix-update/src/...

6. Scroll to the bottom and use this commit message:
   Add V25 partner game playability and real artwork fix

7. Select:
   Commit directly to the main branch

8. Click:
   Commit changes

9. Wait for Vercel to redeploy.

10. Test these URLs:
   https://www.gr8gamz.com/more-free-games/body-drop-3d
   https://www.gr8gamz.com/more-free-games/body-drop-3d/play
   https://www.gr8gamz.com/more-free-games/tentrix
   https://www.gr8gamz.com/more-free-games/tentrix/play
   https://www.gr8gamz.com/api/partner-games/body-drop-3d
   https://www.gr8gamz.com/api/partner-games/tentrix
   https://www.gr8gamz.com/updates/v25-partner-game-playability-real-artwork-fix

11. If the API route returns a resolved game image and URL, the live artwork/play resolver is working.

12. After confirming it works, push indexing signals:
   https://www.gr8gamz.com/api/indexnow/bulk?scope=content&limit=40

Do not request indexing for the /play routes. Request indexing for the profile pages only.
