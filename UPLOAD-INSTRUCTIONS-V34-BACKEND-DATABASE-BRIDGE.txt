GR8 GAMZ V34 upload instructions

Upload the contents of this update pack to the root of your existing V33 GitHub repository.

Correct example paths:

src/app/backend/page.js
src/app/api/gr8/backend/status/route.js
src/lib/server/gr8BackendStore.js
database/v34-gr8-backend-bridge-schema.sql

Do not upload the outer zip folder itself.

Suggested commit message:

Add V34 backend database bridge

After Vercel deploys, test:

https://www.gr8gamz.com/backend
https://www.gr8gamz.com/admin/backend
