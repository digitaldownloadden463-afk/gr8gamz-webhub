# GR8 GAMZ V32 — Arcade Pulse, Missions and Clubhouse Action Layer

## Purpose
V32 makes GR8 GAMZ feel more alive without adding third-party chat, forum or account platforms.

## Added
- `/live` GR8 Arcade Pulse page.
- Homepage Arcade Pulse panel.
- Claimable XP rewards for daily missions.
- Mission claim storage ready for a future in-house database.
- Controlled GR8 Clubhouse room pages:
  - `/community/game-requests`
  - `/community/high-scores`
  - `/community/bug-reports`
  - `/community/favourite-games`
  - `/community/deal-ideas`
- Local moderation queue behaviour for Clubhouse submissions.
- Database schema additions for future mission claims and Clubhouse submissions.
- Sitemap, IndexNow and update post coverage.

## Important product decision
V32 avoids fake public counters. Arcade Pulse is powered by real on-device player actions now and can later be connected to the GR8-owned backend.

## Next phase suggestion
V33 should add the real database/API layer for GR8 Passport, mission claims, Clubhouse submissions and admin moderation.
