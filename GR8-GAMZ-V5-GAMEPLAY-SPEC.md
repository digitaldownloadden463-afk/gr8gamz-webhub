# GR8 GAMZ V5 Premium Gameplay Pass

This update applies the V4 premium gameplay standard to the remaining launch games:

- Stack Tower Rush
- Pixel Goal Hero
- Space Tap Survival

## Shared V5 Standard

Each upgraded game now includes:

- premium thumbnail-backed start screen
- short mobile-first instructions
- larger and clearer CTA buttons
- improved canvas graphics and neon styling
- particle effects and score feedback
- stronger score/best-score tracking
- game-over / result overlays with XP-style rewards
- touch-first controls that also work on desktop
- cache-busted iframe URLs in `src/data/games.json`

## Stack Tower Rush V5

New features:

- neon grid arena
- sharper glowing blocks
- perfect-drop chain system
- perfect-drop particle bursts
- floating feedback text
- new-best celebration
- impactful “Tower collapsed” results screen

Controls:

- Mobile: tap anywhere to drop the moving block
- Desktop: click or press Space

## Pixel Goal Hero V5

New features:

- neon stadium scene
- visible goal and moving keeper
- drag-to-aim shot line
- stronger ball, keeper and goal graphics
- goal burst effects
- streak tracking
- dramatic save/miss result screen

Controls:

- Mobile: drag from the ball and release to shoot
- Desktop: click, drag and release

## Space Tap Survival V5

New features:

- richer space/starfield background
- improved ship rendering
- glowing asteroids
- shield pickups
- boost pickups
- shield-save feedback
- stronger crash/results screen

Controls:

- Mobile: hold the screen to fly upward, release to fall
- Desktop: hold Space or mouse/trackpad
