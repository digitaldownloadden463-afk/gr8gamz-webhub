GR8 GAMZ V33 — Upload Instructions

1. Download and unzip the V33 update pack.
2. Open the GitHub repository:
   github.com/digitaldownloaden463-afk/gr8gamz-webhub
3. Make sure the branch is main.
4. Click Add file → Upload files.
5. Drag in the CONTENTS of the unzipped V33 folder.

Correct paths should look like:

src/app/admin/page.js
src/app/support/page.js
src/components/admin/AdminControlRoom.js
src/lib/moderationClient.js
database/gr8-control-room-schema.sql

Do NOT upload the outer folder itself.

6. Commit directly to main.

Suggested commit message:

Add V33 Control Room moderation and support foundation

7. Wait for Vercel deployment to finish.

8. Test:
