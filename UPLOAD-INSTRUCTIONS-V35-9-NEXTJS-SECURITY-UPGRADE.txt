GR8 GAMZ V35.9 — Upload Instructions

1. Download and unzip the V35.9 update pack.
2. Open GitHub:
   github.com/digitaldownloaden463-afk/gr8gamz-webhub
3. Make sure branch is main.
4. Click Add file → Upload files.
5. Drag the CONTENTS of the unzipped V35.9 folder into the GitHub root.

Correct paths should look like:

package.json
package-lock.json
scripts/audit-v35-9-security-upgrade.mjs
GR8-GAMZ-V35-9-NEXTJS-SECURITY-UPGRADE-DEPENDENCY-LOCK-SPEC.md
UPLOAD-INSTRUCTIONS-V35-9-NEXTJS-SECURITY-UPGRADE.txt

Do not upload the outer folder itself.

Suggested commit message:

Upgrade Next.js security patch and lock dependencies

After Vercel deploys, check the build log confirms:

Detected Next.js version: 14.2.35

Do not run npm audit fix --force.
