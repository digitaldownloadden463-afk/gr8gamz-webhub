
GR8 GAMZ V34 Upload Instructions

1. Download and unzip the V34 update pack.
2. Open github.com/digitaldownloaden463-afk/gr8gamz-webhub
3. Confirm you are on the main branch.
4. Click Add file -> Upload files.
5. Drag the CONTENTS of the unzipped folder, not the outer folder.
6. Correct paths should include:
   - package.json
   - .env.example
   - src/lib/gr8DatabaseCore.server.js
   - src/lib/gr8SyncClient.js
   - src/app/api/gr8/database/status/route.js
   - src/app/admin/database/page.js
   - database/gr8-v34-database-core-schema.sql
7. Commit directly to main.
8. Suggested commit message:
   Add V34 database core and persistent platform API
9. Wait for Vercel to deploy.
10. Test:
   - /admin/database
   - /api/gr8/database/status
   - /api/gr8/passport/sync
   - /updates/v34-database-core-persistent-platform-api

Important:
- The site will still work without DATABASE_URL.
- Database mode starts only after the V34 SQL schema is run and DATABASE_URL or GR8_DATABASE_URL is added in Vercel.
