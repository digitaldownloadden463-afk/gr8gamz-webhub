# GR8 GAMZ V14 Retention + Discovery Engine

## Goal
Turn GR8 GAMZ from a good-looking arcade site into a stronger platform engine that keeps players discovering, playing, saving and returning.

## What V14 adds
- Daily XP reward card on the homepage.
- Local achievement badges.
- Horizontal recommended game rails.
- Quick-play and harder-run discovery sections.
- Improved `/games` filter system.
- Control-type landing pages:
  - `/controls/tap`
  - `/controls/swipe`
  - `/controls/drag`
  - `/controls/hold`
  - `/controls/keyboard`
- Difficulty landing pages:
  - `/difficulty/easy`
  - `/difficulty/medium`
  - `/difficulty/hard`
- Game session tracking improvements.
- Score/result tracking into local player profile when games send results.
- Updated sitemap with control and difficulty pages.

## Why this matters
This phase makes the current game library work harder before adding even more games. The site now has better:
- repeat-visit reasons
- internal linking
- player journey flow
- SEO landing pages
- game discovery paths
- mobile game filtering
